# Implementasi Hybrid System Tabungan

## Tujuan
Memperbaiki sistem tabungan agar terintegrasi penuh dengan konsep **Hybrid System** yang disetujui:
> 1 Sumber Kebenaran = Akun Tabungan Induk + Banyak Target Alokasi

## Konsep Final Yang Disetujui
```mermaid
flowchart LR
    A[Akun Tabungan Induk] --> B[Saldo Total Tabungan]
    B --> C[Target 1: Beli HP]
    B --> D[Target 2: Liburan]
    B --> E[Target 3: Dana Darurat]
    
    C + D + E ≤ B
    
    F[Transaksi Transfer ke Akun Tabungan] --> B
```

✅ **Tidak ada double entry**  
✅ **100% akurat, sinkron dengan transaksi**  
✅ **User tetap dapat progress bar per target**  
✅ **Tidak bisa over-allocate**

---

## 🔹 PERUBAHAN SISTEM LENGKAP

### 1. Perubahan Struktur Data
| Tabel Savings Baru | Keterangan |
|---|---|
| id | Primary key |
| name | Nama target tabungan |
| targetAmount | Nominal target |
| allocatedAmount | Jumlah yang dialokasikan ke target ini |
| accountId | ✅ **BARU**: FK ke akun tabungan induk |
| deadline | Deadline target |
| created_at | Timestamp |

> ⚠️ Tidak ada lagi `currentAmount` manual. Semua berdasarkan alokasi dari saldo akun.

---

### 2. Perubahan Alur Kerja

#### 📌 Alur Input User:
1.  User buat **1 Akun** dengan `type: savings` di halaman Akun
2.  User buat **banyak target tabungan** di halaman Aset & Tabungan
3.  Setiap target akan di-assign ke akun tabungan induk
4.  User melakukan **transfer** dari akun utama ke akun tabungan
5.  Saldo akun tabungan bertambah otomatis
6.  User **alokasikan** berapa persen dari saldo untuk masing-masing target
7.  Jumlah semua `allocatedAmount` **TIDAK BOLEH** melebihi saldo akun tabungan

#### 📌 Yang Terjadi Otomatis:
- ✅ Semua riwayat transfer tercatat di transaksi
- ✅ Saldo akun selalu menjadi 1 sumber kebenaran
- ✅ Tidak perlu input manual tambah saldo target
- ✅ Progress bar diupdate otomatis setiap ada transfer

---

### 3. Perubahan Tampilan Dashboard
Card `Tabungan & Asset` di Dashboard hanya akan menampilkan:
✅ Saldo dari **semua akun bertipe savings**  
✅ Total nilai dari semua aset  
❌ **TIDAK LAGI** mengambil dari tabel savings `currentAmount`

---

### 4. Perubahan Halaman Aset & Tabungan
Halaman Savings section akan menjadi halaman **Alokasi**:
- Menampilkan saldo total akun tabungan di bagian atas
- Daftar target tabungan dengan progress bar alokasi
- Slider untuk membagi-bagi alokasi antar target
- Indikator berapa sisa saldo yang belum dialokasikan
- Tombol "Transfer ke Tabungan" yang langsung buka form transfer
