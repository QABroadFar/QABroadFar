# Plan: Apply Dark Theme UI from Reference

## Goal
Convert the existing Keuangan Keluarga app to use the dark theme design from the HTML reference (Sovereign Ledger).

## Reference Analysis

### Key Design Elements
- **Theme**: Dark mode with Material Design 3 inspired colors
- **Framework**: Tailwind CSS with custom configuration
- **Layout**: Bento grid system (12 columns)
- **Typography**: Manrope (headlines), Inter (body/labels)
- **Icons**: Material Symbols Outlined
- **Primary Color**: `#4edea3` (emerald green)
- **Background**: `#131314` (near black)
- **Cards**: `#1c1b1c` to `#353436` surface variants
