# Plan: Replace Lucide Icons with emoji-mart for Transaction Categories

## Overview
Migrate from Lucide React SVG icons to emoji-mart for category icons. This involves:
- Rewriting IconPicker to use emoji-mart picker
- Converting stored icon data from string identifiers (e.g., "utensils") to emoji characters (e.g., "🍴")
- Updating all UI components that display category icons
- Adding data migration for existing users

## Detailed Steps

```bash
npm install @emoji-mart/react @emoji-mart/data
```

### 4. Add Data Migration
**File:** `src/context/AppContext.jsx`

Import `migrateIcon` and modify categories state initialization:

```js
import { migrateIcon } from '../utils/iconMigration';

const [categories, setCategories] = useState(() => {
  const stored = storage.get('categories', defaultCategories);
  let needsMigration = false;
  const migrated = stored.map(cat => {
    const newIcon = migrateIcon(cat.icon);
    if (newIcon !== cat.icon) needsMigration = true;
    return { ...cat, icon: newIcon };
  });
  if (needsMigration) storage.set('categories', migrated);
  return migrated;
});
```
